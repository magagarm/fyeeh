Безплатні хости — Кроки

GitHub Pages

Публікує статичні сайти з репозиторію GitHub. Підтримує автоматичну збірку через GitHub Actions або пряме розгортання з гілки.

1) Створіть репозиторій на GitHub. 2) Завантажте index.html (push). 3) У Settings → Pages оберіть гілку (main) як джерело. 4) Сайт буде доступний за https://USERNAME.github.io/REPO.

Netlify

Підключіть репозиторію або перетягніть папку з готовими файлами (Netlify Drop). Має безкоштовний тариф з CI/CD.

1) Увійдіть на netlify.com і натисніть "Add new site" → "Import from Git" або "Drag & Drop". 2) Дайте дозвіл на доступ до репозиторію. 3) Налаштуйте build command (якщо потрібен) і publish directory (наприклад, /). 4) Netlify створить live сайт з власним доменом, наприклад https://glittering-moth-123.netlify.app

Vercel

Добре підходить для сучасних фреймворків; дуже просте підключення репозиторію та CLI для швидкого деплою.

1) Зареєструйтесь на vercel.com та підключіть GitHub. 2) Імпортуйте репозиторій (Add New Project). 3) Налаштуйте (Vercel автоматично вгадує framework). Натисніть Deploy. 4) Отримаєте адресу виду https://your-site.vercel.app
